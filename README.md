# ❤️ Heart Disease Data Analysis using Tableau
---

## 📌 Project Overview

Heart disease remains one of the leading causes of mortality worldwide. Lifestyle changes, poor diet, and lack of physical activity contribute significantly to cardiovascular diseases.

This project analyzes a **Heart Disease dataset from Kaggle** using **Tableau** to uncover important patterns and risk factors. The data is transformed into interactive dashboards that help identify high-risk groups and support better healthcare decision-making.

The analysis focuses on factors such as:

- Age
- Gender
- Cholesterol Levels
- Blood Pressure
- Lifestyle Habits
- Medical Indicators

## 🎯 Objectives

The main goals of this project are:

- Analyze heart disease data using visualization techniques
- Identify major risk factors contributing to heart disease
- Compare patient groups based on age, gender, and medical indicators
- Create interactive dashboards using Tableau
- Present insights that support preventive healthcare decisions

---

## 📂 Dataset

Dataset used in this project:

Heart Disease UCI Dataset (Kaggle)

Features in the dataset include:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Number of Major Vessels
- Thalassemia
- Target (Heart Disease Presence)

## 📊 Dashboard Features

The Tableau dashboard provides the following visual insights:

- Heart disease distribution by **age group**
- Comparison of **male vs female patients**
- Relationship between **cholesterol and heart disease**
- Analysis of **blood pressure and heart risk**
- Identification of **high-risk patient groups**

---

## 📈 Key Insights

Some important insights discovered during the analysis:

- Middle-aged individuals show higher risk of heart disease
- Higher cholesterol levels correlate with increased risk
- Male patients show slightly higher prevalence
- Lifestyle factors significantly impact heart health
